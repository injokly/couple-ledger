# Spec 07 — 거래 내역 화면

## 목표

과거 거래를 조회, 필터링, 편집, 삭제한다.

## 의존성

- `03-transactions.md` 완료

## 라우트

- `/transactions` — 리스트
- `/transactions/:id` — 상세/편집 (모달 or 별도 페이지)

## 화면 구성

### 상단 고정 헤더

```
┌─────────────────────────────────┐
│  거래 내역                      │
│  2026년 4월  ‹ ›                │
├─────────────────────────────────┤
│  수입 12,400,000  지출 4,850,000│
│  저축 7,550,000 · 60.9%        │
├─────────────────────────────────┤
│  [전체] [지출] [수입] [이체]  🔍│
└─────────────────────────────────┘
```

- 월 네비게이션 (‹, ›)
- 월별 요약 (수입/지출/저축)
- 타입 필터 탭
- 검색 아이콘 (탭하면 검색 모달)

### 리스트

날짜 그룹별로 묶기:

```
4월 21일 (오늘) · 지출 2건                    ₩94,800
──────────────────────────────────
🍜 점심 식사                 −12,500원
   식비 · 인조 · 12:48                  

🛒 이마트 장보기             −82,300원
   식비 · 인조 · 어제 19:34

4월 20일 (어제) · 수입 1건                 +6,500,000
──────────────────────────────────
💼 4월 월급                +6,500,000원
   급여 · 수진 · 09:02
```

### 거래 행 인터랙션

- 탭 → 상세/편집 모달 열기
- 좌측 스와이프 → 삭제 (확인 후)
- 길게 누르기 → 선택 모드 (여러 건 삭제)

## 편집 모달

빠른입력 모달과 구조 동일하되:
- 저장 버튼 아래 "삭제하기" (빨간 텍스트 버튼)
- 타입은 비활성화 (변경 불가, 수정 필요시 삭제 후 재생성)
- 나머지 필드 모두 편집 가능

## 검색 모달

- 전체 화면 오버레이
- 상단 검색 입력 (autofocus)
- 실시간 결과 (debounce 300ms)
- 검색 대상: 메모, 카테고리명, 태그
- 최근 검색어 저장 (최대 10개, localStorage)

## 필터 (상세)

검색 모달 내에서 필터 UI 제공:
- 기간 (오늘 / 이번 주 / 이번 달 / 지난 달 / 3개월 / 사용자 지정)
- 카테고리 (다중 선택)
- 계좌 (다중 선택)
- 입력자 (인조 / 수진 / 전체)
- 금액 범위

// 무한 스크롤

## 수용 기준

- [ ] 기본 보기: 이번 달 전체 거래, 최신 순
- [ ] 월 네비게이션으로 과거 월 조회
- [ ] 월별 요약 카드 정확 (수입/지출/저축률)
- [ ] 타입 필터 탭 동작
- [ ] 날짜 그룹 헤더 (오늘/어제/N일 전/날짜)
- [ ] 그룹별 일일 합계 (소액 빠르게 파악)
- [ ] 거래 탭 → 편집 모달
- [ ] 좌측 스와이프 → 삭제 확인 → 삭제
- [ ] 검색 모달 실시간 동작 (debounce 300ms)
- [ ] 스크롤 하단 도달 시 다음 50건 로드 (무한 스크롤)
- [ ] 검색 결과 없음 상태 UI
- [ ] 거래 없음 상태 (이번 달 0건) → "+ 거래 추가" CTA
- [ ] optimistic delete (탭 즉시 리스트에서 제거)
- [ ] 편집 후 저장 시 `updated_at` 갱신 표시 ("수정됨" 아이콘)

## 개방 질문

1. 삭제 후 "되돌리기" 토스트? (5초간 취소 가능)  
   → **제안: 포함. 실수 방지에 유용.**
2. 월별 요약의 "저축률"은 수입 0일 때 표시?  
   → **수입 0이면 "—" 표시.**
3. CSV 내보내기 이 화면에 둘까?  
   → **Phase 2. 설정으로.**
